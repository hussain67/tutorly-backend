# Tutorly Backend

Initial scaffolding for the Tutorly project backend.

## Overview

Tutorly is an online learning platform where instructors can create courses and students can subscribe.  
This repository contains the backend code including API endpoints, database models, and server setup.

## Tech Stack

- Node.js + Express
- MongoDB
- TypeScript
- Supertest
- GitHub Actions for CI/CD
